<task id="5">
Add image placeholder component for hero:
- Dashed border placeholder
- Gray background
- "Your photo here" text
- Accepts `src` prop for when image is provided
- Falls back to placeholder when no src

```tsx
function ImagePlaceholder({
  src,
  alt,
  className
}: {
  src?: string;
  alt: string;
  className?: string;
}) {
  if (src) {
    return (
      <Image
        src={src}
        alt={alt}
        fill
        className={cn("object-cover", className)}
      />
    );
  }

  return (
    <div className={cn(
      "flex items-center justify-center bg-muted/50 border-2 border-dashed border-border rounded-lg",
      className
    )}>
      <span className="text-sm text-muted-foreground">Your photo here</span>
    </div>
  );
}
```
</task>
