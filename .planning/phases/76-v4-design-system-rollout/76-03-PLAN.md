# Plan 76-03: Redesign Reputation Page with V4 Design Language

## Objective

Rebuild the `/reputation` page using the V4 design language — dark/light alternating sections, oversized typography, Framer Motion scroll animations, floating shapes, pill CTAs, and the shared V4 components. Preserve all SEO metadata and structured data.

## Context

The current `/reputation` page imports 10 V2 components and stacks them. It has strong SEO (FAQ schema, Open Graph, keywords). The redesign keeps all content and SEO but completely changes the visual presentation to match the V4 style from the homepage.

The reputation page is about the **Review Add-On** ($99/mo) and positions AvisLoop's automated review management service. It uses the shared `(marketing)` layout (nav + footer from layout.tsx).
