# Plan 76-05: Update Marketing Layout + Legal Pages Typography

## Objective

Update the shared `(marketing)` layout's nav and footer to match the V4 design language. Improve the legal pages (privacy, terms, SMS compliance) with better typography and spacing — without changing any content.

## Context

The `(marketing)` layout wraps: `/reputation`, `/pricing`, `/client-portal`, `/privacy`, `/terms`, `/sms-compliance`, and `/audit`. The homepage is now in `(home)` and has its own layout.

Legal pages (privacy, terms, SMS) are 300-400 lines of compliance text. They should NOT be redesigned — the content is legally required. But the typography, spacing, and readability can be improved.

## Tasks

<task id="1">
Update `app/(marketing)/layout.tsx` nav to V4 style:

Replace the current navbar with V4-style navigation:
- Transparent → frosted glass on scroll (same pattern as V4Nav)
- Logo with accent-colored logomark
- Desktop links: ghost text links (not buttons), matching V4 style
- Links: Reputation, Pricing, Client Portal
- Right side: theme switcher + auth CTA (keep existing `AuthButtons` Suspense pattern)
- Mobile: hamburger → full-screen overlay menu (same pattern as V4Nav mobile menu)

The nav must be a `'use client'` component. Extract it if needed — the layout can import it. Keep the `AuthButtons` server component with Suspense wrapping.

**Key difference from homepage nav:** This nav links to actual pages (`/reputation`, `/pricing`), not hash anchors. Adjust the links array accordingly.
</task>

<task id="2">
Update the footer in `app/(marketing)/layout.tsx` to V4 style:

Replace the current 4-column grid footer with the V4 minimal footer:
- Single row: logo + copyright left, links right
- Links: Privacy, Terms, Reputation, Client Portal
- Subtle border-top separator
- Same muted color treatment as the homepage footer
</task>

<task id="3">
Improve legal page typography. For `privacy/page.tsx`, `terms/page.tsx`, and `sms-compliance/page.tsx`:

Do NOT change any text content. Only apply styling improvements:

- Wrap the prose in a `max-w-3xl mx-auto` container (narrower for readability)
- Add `prose`-like styling via Tailwind classes:
  - Headings (h2): `text-2xl font-bold mt-12 mb-4`
  - Headings (h3): `text-lg font-semibold mt-8 mb-3`
  - Paragraphs: `text-muted-foreground leading-relaxed mb-4`
  - Lists: `space-y-2 text-muted-foreground`
  - Strong/bold: already handled by HTML
- Add a styled header at the top with the page title + effective date
- Add a "Back to home" link at the top using the AccentBar + breadcrumb pattern

This is purely a typography pass — no content changes, no section removals.
</task>
