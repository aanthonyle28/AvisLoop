**Part A: DB baseline — Run the analytics RPC directly to get expected values**

Execute via Supabase MCP:

```sql
SELECT
  j.service_type,
  COUNT(sl.id) AS total_sent,
  COUNT(sl.id) FILTER (WHERE sl.status IN ('sent', 'delivered', 'opened')) AS delivered,
  COUNT(sl.id) FILTER (WHERE sl.reviewed_at IS NOT NULL) AS reviewed,
  COUNT(DISTINCT cf.id) AS feedback_count
FROM jobs j
LEFT JOIN campaign_enrollments ce ON ce.job_id = j.id
LEFT JOIN send_logs sl ON sl.campaign_enrollment_id = ce.id
LEFT JOIN customer_feedback cf ON cf.customer_id = j.customer_id
  AND cf.business_id = '6ed94b54-6f35-4ede-8dcb-28f562052042'
WHERE j.business_id = '6ed94b54-6f35-4ede-8dcb-28f562052042'
GROUP BY j.service_type
ORDER BY COUNT(sl.id) DESC;
```
