<task type="auto">
  <name>Task 1: Create BusinessSwitcher component</name>
  <files>components/layout/business-switcher.tsx</files>
  <action>
Create a new client component `components/layout/business-switcher.tsx` that:

1. **Reads from context:** Call `useBusinessSettings()` to get `businesses`, `businessId`, and `businessName`. Do NOT accept these as props.

2. **Single-business guard:** When `businesses.length <= 1`, render ONLY a plain `<span>` with the business name (truncated via `truncate` class). No chevron. No dropdown. No interactive affordance. This is the most common case.

3. **Multi-business dropdown:** When `businesses.length > 1`, render a Radix `DropdownMenu` (imported from `@/components/ui/dropdown-menu`):
   - **Trigger:** A `<button>` styled as a clickable row with:
     - The current `businessName` (truncated via `truncate` class)
     - A `CaretUpDown` icon from `@phosphor-icons/react` (size 14, `text-muted-foreground`, `shrink-0`)
     - Hover: `hover:bg-secondary/70 dark:hover:bg-muted/70 transition-colors`
     - Pending: `opacity-60 pointer-events-none` when `isPending` is true
     - `aria-label` that reads: `Current business: ${businessName}. Click to switch.`
   - **Content:** `DropdownMenuContent` with `align="start"` and `className="w-56"`:
     - Map over `businesses` to render `DropdownMenuItem` for each
     - Each item shows the business name (truncated) + a `Check` icon (Phosphor, size 14, `text-accent`) when `biz.id === businessId`
     - `onSelect` calls `handleSelect(biz.id)`

4. **Switch handler:** `handleSelect(id: string)` should:
   - Early return if `id === businessId` (already selected) or `isPending` (in flight)
   - Call `startTransition(async () => { const result = await switchBusiness(id); if (result?.error) toast.error('Failed to switch business') })`
   - Do NOT call `router.refresh()` or `router.push()` — `revalidatePath` inside `switchBusiness` handles the re-render

5. **useTransition:** Use `const [isPending, startTransition] = useTransition()` (React import, not a custom hook).

6. **Imports:**
   - `useTransition` from `react`
   - `CaretUpDown`, `Check` from `@phosphor-icons/react`
   - `DropdownMenu`, `DropdownMenuContent`, `DropdownMenuItem`, `DropdownMenuTrigger` from `@/components/ui/dropdown-menu`
   - `useBusinessSettings` from `@/components/providers/business-settings-provider`
   - `switchBusiness` from `@/lib/actions/active-business`
   - `cn` from `@/lib/utils`
   - `toast` from `sonner`

7. **Export:** Named export `BusinessSwitcher`.

Follow the exact pattern from `account-menu.tsx` for DropdownMenu usage.
