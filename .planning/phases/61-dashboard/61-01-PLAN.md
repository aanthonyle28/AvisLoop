1. `mcp__supabase__execute_sql`: Get the business_id for the test account:
```sql
SELECT b.id, b.name

```
Record the business_id and business name.

2. `mcp__supabase__execute_sql`: Get review count this month:
```sql
SELECT COUNT(*) as reviews_this_month
FROM send_logs
WHERE business_id = '[business_id from step 1]'
  AND reviewed_at IS NOT NULL
  AND created_at >= date_trunc('month', NOW());
```

3. `mcp__supabase__execute_sql`: Get average rating:
```sql
SELECT COALESCE(ROUND(AVG(rating)::numeric, 1), 0) as avg_rating
FROM customer_feedback
WHERE business_id = '[business_id]'
  AND submitted_at >= date_trunc('month', NOW());
```

4. `mcp__supabase__execute_sql`: Get sent count and conversion rate components:
```sql
SELECT
  COUNT(*) FILTER (WHERE status IN ('sent','delivered','opened') AND created_at >= NOW() - INTERVAL '7 days') as sent_this_week,
  COUNT(*) FILTER (WHERE status IN ('sent','delivered','opened') AND created_at >= date_trunc('month', NOW())) as sent_this_month,
  COUNT(*) FILTER (WHERE reviewed_at IS NOT NULL AND created_at >= date_trunc('month', NOW())) as reviewed_this_month
FROM send_logs
WHERE business_id = '[business_id]';
```

5. `mcp__supabase__execute_sql`: Get active enrollments and pending sends:
```sql
SELECT
  (SELECT COUNT(*) FROM campaign_enrollments WHERE business_id = '[business_id]' AND status IN ('active', 'frozen')) as active_sequences,
  (SELECT COUNT(*) FROM send_logs WHERE business_id = '[business_id]' AND status = 'pending') as pending_sends;
```

6. `mcp__supabase__execute_sql`: Get job counts for queue context:
```sql
SELECT COUNT(*) as total_jobs FROM jobs WHERE business_id = '[business_id]';
```

7. `mcp__supabase__execute_sql`: Get attention alert counts:
```sql
SELECT
  (SELECT COUNT(*) FROM send_logs WHERE business_id = '[business_id]' AND status IN ('failed', 'bounced') AND (error_message IS NULL OR error_message NOT LIKE '[ACK]%')) as failed_sends,
  (SELECT COUNT(*) FROM customer_feedback WHERE business_id = '[business_id]' AND resolved_at IS NULL) as unresolved_feedback;
```

8. `mcp__supabase__execute_sql`: Get recent activity event count:
```sql
SELECT COUNT(*) as recent_events FROM (
  SELECT created_at FROM send_logs WHERE business_id = '[business_id]' AND campaign_id IS NOT NULL
  UNION ALL
  SELECT submitted_at FROM customer_feedback WHERE business_id = '[business_id]'
  UNION ALL
  SELECT enrolled_at FROM campaign_enrollments WHERE business_id = '[business_id]'
  ORDER BY 1 DESC LIMIT 5
) sub;
```

Record ALL SQL results. These are ground truth for KPI comparison.
